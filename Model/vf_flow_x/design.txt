数据集包含一条或多条记录，
记录在内存中存储数据，
每条记录的数据结构可以自定义

当数据需要在记录间分析、传输时，
需要使用映射器（Mapper）
Mapper通过Map方法进行记录数据映射
映射器可以通过数据脚本驱动

	void Record::Deliver(Record* who)
	{
		if (_Data == null)
			return;

		if (_Access == VF_ACCESS_IN 
			|| who->_Access == VF_ACCESS_OUT)
			return;

		who->Write(_Data, _Size);
	}

在流模型里链接存储记录之间的数据脚本
用于自动化数据映射

执行算法
节点之间通过链接连接
节点本身可以执行，但是不直接影响其他节点的执行
节点执行后进行链接的映射

执行通过执行引擎进行
链接末端的节点通过链接约束确定是否能够执行
每次映射完成后，会自动调用节点的约束验证
验证通过即可执行，可执行则排入执行队列

没有入链的节点没有约束，直接排入队列

引擎即时扫描队列，发现节点即执行
节点没有Start方法，只有Run方法
Run是同步方法，返回即表示执行完成
节点使用Context进行状态监控
